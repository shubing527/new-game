溫習：電解質與水溶液導電

本份重點直接對應 50 題題庫，建議先讀概念，再看「對應題型」核對自己會不會。

① 定義 ② 強/弱/非 ③ 酸鹼鹽 ④ 生活水溶液 ⑤ 固體與熔融 ⑥ 影響因素 ⑦ 應用與安全 ⑧ 常見迷思 ⑨ 對應題型

① 核心定義

  • 電解質：在水溶液中或熔融狀態下能解離出可自由移動的離子，使溶液能導電的物質。
  • 導電的本質：陰陽離子分別向相反電極移動形成電流，並非水分子本身在導電。
  • 純水導電性極差；自來水可導電是因為含有微量礦物質離子。
  • 固態的離子化合物（例如固體 NaCl）雖然有離子，但離子被晶格鎖住不能移動，所以不導電。

② 強電解質 / 弱電解質 / 非電解質

類別解離程度常見例子
強電解質幾乎完全解離NaCl、KCl、HCl、HNO₃、H₂SO₄、NaOH、KOH
弱電解質只有少部分解離醋酸 (CH₃COOH)、檸檬酸、氨水 (NH₃·H₂O)、碳酸 (H₂CO₃)
非電解質幾乎不解離糖（蔗糖、葡萄糖）、酒精（乙醇）、油類、純水、丙酮

小訣竅：能拆成「正離子＋負離子」的，幾乎都會導電；只是「整顆分子溶解」的（如糖、酒精），通常不導電。

③ 酸、鹼、鹽各舉幾個

  • 強酸：鹽酸 HCl、硫酸 H₂SO₄、硝酸 HNO₃。
  • 弱酸：醋酸、碳酸、檸檬酸（檸檬汁酸味來源）。
  • 強鹼：氫氧化鈉 NaOH、氫氧化鉀 KOH。
  • 弱鹼：氨水 NH₃·H₂O、小蘇打水 (NaHCO₃ 偏弱鹼性，本身是鹽類)。
  • 鹽類：NaCl、KCl、CaCl₂、NaHCO₃、Na₂SO₄ — 溶於水大多是強電解質。
  • 酸＋鹼中和會生成鹽＋水，例如 HCl + NaOH → NaCl + H₂O，溶液中仍有 Na⁺、Cl⁻ 故仍可導電。

④ 生活中的水溶液：會不會導電？

溶液導電？原因
食鹽水 / 海水會Na⁺、Cl⁻ 完全解離；海水離子濃度高，導電性更強
運動飲料 / 口服電解質液 (ORS)會含 Na⁺、K⁺、Cl⁻ 等電解質補充流失
自來水會（弱）含微量礦物質離子
食醋 / 檸檬汁會（弱）弱酸只部分解離，導電性比鹽水弱
小蘇打水會NaHCO₃ 是鹽類，解離出 Na⁺、HCO₃⁻
汗水 / 體液會含 Na⁺、K⁺、Cl⁻ 等
洗潔劑 / 肥皂水通常會含離子型界面活性劑與電解質；純非離子型則弱
糖水 / 蔗糖溶液不會糖分子整顆溶解，不解離出離子
酒精（乙醇）/ 酒精消毒液不會乙醇是非電解質
食用油 / 礦物油不會非極性、不解離，也不溶於水
蒸餾水 / 純水幾乎不會幾乎沒有離子
雨水很弱溶有微量 CO₂、SO₂ 等，導電性接近純水

⑤ 固體、水溶液與熔融狀態

  • 固體 NaCl：離子被鎖在晶格中 → 不導電。
  • NaCl 水溶液：解離成 Na⁺(aq)、Cl⁻(aq) → 能導電。
  • 熔融 NaCl：高溫熔成液體後離子可自由移動 → 能導電（電解工業常見）。
  • 把食鹽倒入水中是物理性溶解，水分子的極性把 Na⁺、Cl⁻ 拉開（水合）。
  • 冰塊本身（即使原本是鹽水結成）導電性極差，因為離子又被固定住了。

⑥ 影響導電性強弱的因素

  • 離子濃度：濃度越高，可移動的離子越多 → 導電性越強（到飽和前）。
  • 溫度：溫度升高，離子運動更快、解離更完全 → 導電性通常變強。
  • 離子種類與電荷：電荷數越大、遷移率越高的離子貢獻越大。
  • 強電解質 vs 弱電解質：同濃度下強電解質導電性遠大於弱電解質。
  • 把糖加進鹽水裡，鹽水的導電性幾乎不變（糖不解離）。

⑦ 應用與安全

  • 電池：內部使用電解質（液態、膠態或固態）讓離子在正負極之間搬運電荷。
  • 電鍍 / 電解：在電解質溶液中通電，使金屬離子在電極上還原成金屬層；電解水也需加少量電解質才導電。
  • 導電度計 / TDS 計：量測水中可溶解固體（離子）含量，數值越高代表離子越多。
  • 醫學：腹瀉脫水時要補充電解質（ORS）；只喝純水反而稀釋體內鈉離子。
  • 安全：潮濕又有鹽分的表面（汗手、潮濕地板）導電性大增，絕對不要在這種狀態下碰電器或插座。

⑧ 題目常見的「誤解澄清」

  • 「液體就會導電」❌ — 油、酒精、糖水都是液體但不導電。
  • 「透明溶液一定不導電」❌ — 食鹽水、稀鹽酸都透明，但是強電解質。
  • 「有機物都是非電解質」❌ — 醋酸、檸檬酸是有機物但是弱電解質。
  • 「水自己會導電」❌ — 純水導電性極弱，導電的是溶在水中的離子。
  • 「糖水加鹽不會變化」❌ — 加鹽會大幅提高導電性，糖則沒貢獻。
  • 「鹽固體會導電」❌ — 固體不導電，要「溶於水」或「熔融」才行。

⑨ 對應題型（常見考法）

  1. 定義題：哪一個敘述最能描述「電解質」？— 對應第 1、2 題。
  2. 分類題：把 NaCl、糖、醋酸、NaOH 分到強/弱/非電解質。— 對應第 3–8 題。
  3. 生活情境：拿燈泡電路測哪杯液體會亮？— 海水、運動飲料、糖水、酒精、油、蒸餾水都常考。— 對應第 10–15、22、29 題。
  4. 狀態題：固體 NaCl vs 水溶液 vs 熔融 NaCl 誰能導電？— 對應第 14、15 題。
  5. 影響因素：濃度升高、加熱、加糖、加更多鹽水會怎樣？— 對應第 16、17 題。
  6. 應用題：電池、電鍍、電解水、TDS 計的原理。— 對應第 25–28、34 題。
  7. 安全題：濕手、潮濕鹽分表面為什麼更容易觸電？— 對應第 13 題。
  8. 迷思澄清：「水自己會導電」、「液體就會導電」哪個錯？— 對應第 49、50 題。
